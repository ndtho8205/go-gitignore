# go-gitignore

Generate a .gitignore file from the command line using gitignore.io API.

Inspired by [gitignore.io](https://www.gitignore.io/) and [vue-cli](https://github.com/vuejs/vue-cli)
```shell
ggi
ggi create linux,android
ggi list
ggi search
```
